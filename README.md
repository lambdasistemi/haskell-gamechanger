# haskell-gamechanger

Haskell client for the [GameChanger](https://gamechanger.finance/) Cardano wallet.

Status: **understanding phase** — repository is being bootstrapped with
protocol documentation, an RDF ontology browsable via
[graph-browser](https://github.com/lambdasistemi/graph-browser), and a
security model. Haskell implementation will follow once the constitution
is stable.
